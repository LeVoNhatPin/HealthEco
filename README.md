# HealthEco - Healthcare Appointment System

A modern healthcare appointment booking system built with .NET 8 and Next.js 14.

## Features

### For Patients
- ✅ Book appointments with doctors/clinics
- ✅ Online consultation via video call
- ✅ Medical records management
- ✅ Prescription and medication tracking
- ✅ Review and rating system

### For Doctors
- ✅ Schedule management
- ✅ Patient queue management
- ✅ Electronic medical records
- ✅ Prescription templates
- ✅ Telemedicine consultations

### For Clinics
- ✅ Multi-doctor management
- ✅ Appointment scheduling
- ✅ Billing and payment processing
- ✅ Analytics and reporting
- ✅ Staff management

## Tech Stack

### Backend
- .NET 8 (Clean Architecture)
- PostgreSQL with Entity Framework Core
- JWT Authentication
- SignalR for real-time features
- Redis for caching
- Serilog for logging

### Frontend
- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- Radix UI components
- React Query
- Zustand for state management

## Project Structure
